
# DevilSoulTextPortrait

A fullscreen CSS text portrait project where text is clipped over a background image, creating a dark, stylish visual effect. Inspired by the “Devil Soul” concept, it combines typography, background images, and modern CSS techniques.

## Features

- Fullscreen text portrait on any screen size
- Background image clipped inside text
- Responsive design for laptops and desktops
- Scrollable text inside the portrait
- Stylish dark theme
